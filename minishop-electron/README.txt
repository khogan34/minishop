# MiniShop Electron App (Windows)

## How to run
1. Install Node.js (v18+ recommended)
2. Open terminal in this folder
3. Install dependencies:
   npm install
4. Run development mode (hot reload):
   npm run electron:dev
5. Build production executable (.exe):
   npm run electron:build
   -> Check the "release" folder for the portable EXE

Make sure to run "npm run build" before packaging if you change the app code.
